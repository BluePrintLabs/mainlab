Question: What is HTML?
Answer: HTML stand for Hyper Text Markup Language. It's the foundational code that runs a web page. I often refer to it as the Foundation of your house. With a combination of tags you can design the base layout of any web page. and use it locally, or reference our CSS CDN.
Q: So what does it look like how do I start?
A: Luckily for you, HTML is accepted in most browsers and on most computers. It's important to keep in mind any website or webapp is just a collection of hosted files. So in order to start building with HTML you'll need....
Q: That's right a hosted HTML file!
A: You can host it locally or in the cloud onGithub. Yes, don't over complicate it. All you need is a file that's stored as .html instead of something like .txt or .csv So break out your textpad/notepad on your computer, or your favorite code editor and save a file as (index.html). Why did we name it index instead of something like webpage.html? Well you always start with your homepage otherwise known as your index page. Move onto the next page if you have your index.html file open and ready to start editing.
