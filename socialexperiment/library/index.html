Question: What is a CSS Library?
Answer: In simple terms, it's just a bunch of CSS. We've built out CSS Classes for other people to reference. You can download the file and use it locally, or reference our CSS CDN.
Q: Why do people make CSS Libraries?
A: Librarys are common in many types of computer languages. For example the Bootstrap Library is extermely useful to refernce when building mobile first websites. They have compiled unique mobile desing for you to use instead of designing each style class yourself.
Q: What is special about this CSS Library?
A: Our goal is to compile Low Poly desing for CSS. Many unique libraries are out there for fancy design and text animation, however we have yet to come across a Low Poly one.Library.
